About Bcrypt

                            
Bcrypt is a password hashing function designed to be slow and computationally expensive, making it resistant to brute-force attacks. The "rounds" parameter determines how many times the hash function is applied - higher values are more secure but slower.

                            
Recommended rounds: 10-12 for most applications
